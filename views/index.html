API Basejump: Image Search Abstraction Layer

Free Code Camp Backend Challenge by Mahesh

User Stories:
  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search string.

Example: Search for Images

{ https://image-search-mahesh.glitch.me/imageSearch/elephants?offset=2 }

Usage

Replace "elephants" with image search name and offset = page number

Example: Your Latest Searches

{ https://image-search-mahesh.glitch.me/recentSearches/ }
